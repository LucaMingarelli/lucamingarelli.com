Adjacency matrix for Square Lattice

Given a \(n\times n\) square lattice with elements indexed by \((i,\ j)\) it is useful to be able to quickly generate the associated adjacency matrix.
First of all, notice that the adjacency matrix will be of size \(n^2\times n^2\).
