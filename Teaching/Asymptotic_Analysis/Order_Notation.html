Asymptotic Analysis: Order Notation

Consider two functions \(f, g: \mathbb{C}\rightarrow\mathcal{D}\subset \mathbb{C}\) such that the limit for \(z\rightarrow z_0\) exists in some complex domain \(\mathcal{D}\).

Definition 1 — Asymptotic boundedness
If there exists constants \(K\ge 0\) and \(\delta>0\) such that \(|f(z)|\le K|g(z)|\) for \(0<|z-z_0|<\delta\), then we say \(f(z)\) is asymptotically bounded by \(g(z)\) as \(z\rightarrow z_0\), and we write \begin{equation} f(z) = \mathcal{O}(g(z))\quad \text{as}\quad z\rightarrow z_0. \end{equation} This implies: \[ \left|\frac{f(z)}{g(z)}\right|<\infty. \]


Definition 2 — Asymptotic smallness
If for all \(\epsilon>0\) there exist some constant \(\delta>0\) such that \(|f(z)|\le \epsilon|g(z)|\) for \(0<|z-z_0|<\delta\), then we say \(f(z)\) is asymptotically smaller than \(g(z)\) as \(z\rightarrow z_0\), and we write \begin{equation} f(z) = \mathcal{o}(g(z))\quad \text{as}\quad z\rightarrow z_0, \end{equation} or equivalently \(f(z) \ll g(z)\ \text{as}\ z\rightarrow z_0\). This implies: \[ \frac{f(z)}{g(z)}\xrightarrow[]{z\rightarrow z_0}0. \]


Definition 3 — Asymptotic Equality
If in the limit \(z\rightarrow z_0\) one has \(f(z)=g(z)+\mathcal{o}(g(z))\) then we say \(f(z)\) is asymptotically equivalent to \(g(z)\) as \(z\rightarrow z_0\), and we write \begin{equation} f(z) \sim g(z)\quad \text{as}\quad z\rightarrow z_0. \end{equation} This implies: \[ \frac{f(z)}{g(z)}\xrightarrow[]{z\rightarrow z_0}1. \]
